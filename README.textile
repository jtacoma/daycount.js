h1. JavaScript Calendars

This is an effort to develop some JavaScript modules for calendar and date calculations.

h2. Usage

The *calendars.js* module is published through "GitHub Pages" CDN so you can link to it there:

bc. http://yellowseed.org/javascript-calendars/js/calendars.js

See the "demo":http://yellowseed.org/javascript-calendars for more information about how to use *calendars.js*.

h2. Development

Clone the GitHub repository:

bc. git clone git://github.com/jtacoma/jtacoma.github.com.git

See the "wiki":http://github.com/jtacoma/javascript-calendars/wiki for more information.
