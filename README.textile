h1. JavaScript Calendars

This is an effort to develop some JavaScript modules for calendar and date calculations, together with a useful "demo":http://yellowseed.org/javascript-calendars.

h2. Usage

The `calendars.js` library is published through "GitHub Pages":http://pages.github.com/ at "yellowseed.org/javascript-calendars/lib/calendars.js":http://yellowseed.org/javascript-calendars/lib/calendars.js.

See the "wiki":https://github.com/jtacoma/javascript-calendars/wiki for more information.

h2. Development

Clone the GitHub repository:

<pre>
<code>
git clone git://github.com/jtacoma/jtacoma.github.com.git
</code>
</pre>

*javascript-calendars* uses the cross-vm "Jasmine":http://pivotal.github.com/jasmine/ testing framework.
