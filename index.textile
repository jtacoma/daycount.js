h1. calendar.js

This is a demonstration of the <a href="https://github.com/jtacoma/calendar.js">calendar.js</a> project, a library for working with calendars.

For example, here is some simple information about today:

bq. <div id="sample" />

<script type="text/javascript">
  var today = new calendars.day();
  $('#sample').html(
    'gregorian: ' + today.gregorian + '<br />' + 
    'julianDay: ' + today.julianDay + '<br />' +
    'dreamspell: ' + today.dreamspell);
</script>

To put this same information on another page, copy and paste the following code:

bc. <script type="text/javascript" src="http://yellowseed.org/calendar.js/js/jquery.min.js"></script>
<script type="text/javascript" src="http://yellowseed.org/calendar.js/js/calendars.js"></script>
<div id="sample" />
<script type="text/javascript">
  var today = new calendars.day();
  $('#sample').html(
    'gregorian: ' + today.gregorian + '<br />' + 
    'julianDay: ' + today.julianDay + '<br />' +
    'dreamspell: ' + today.dreamspell);
</script>

While you can paste the whole block in one place, you may want to put the first two @<script>@ tags inside @<head>@.  You don't even need the first line if you're already pulling in "jQuery":http://jquery.com from elsewhere.
