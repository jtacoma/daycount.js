h1. daycount.js

This is a demonstration of the <a href="https://github.com/jtacoma/daycount.js">daycount.js</a> project.

For example, here is some information about today:

bq. <div id="sample" />

<script type="text/javascript">
  var today = new daycount.day();
  $('#sample').html(
    'gregorian: ' + today.gregorian + '<br />' + 
    'julianDay: ' + today.julianDay + '<br />' +
    'dreamspell: ' + today.dreamspell);
</script>

This information was calculated by your browser, on your machine, in your time zone.  To add this to your page, copy and paste the following code into the @<body>@ of any html file:

bc. <script type="text/javascript" src="http://yellowseed.org/daycount.js/js/jquery.min.js"></script>
<script type="text/javascript" src="http://yellowseed.org/daycount.js/js/daycount-0.0.js"></script>
